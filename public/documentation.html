My Flix API

About MyFlix Application

This web application is designed for movie enthusiasts who enjoy information reading information about differente movies and frontend developers who will work on the client-side applications.
The users will be able to use the myFlix application whenever they like to read information about different movies or update their user information, for instance, their list of "Favourite Movies".
This application itself is responsive and can therefore be used anywhere and on any device, giving all users the same experience.
Movie enthusiasts will be able to access information about different movies, directors, and genres. User will be able to sign up, update their personal information, and create a list of their favourite movies.
The server-side of the myFlix application will ensure they have access to this information, that their requests can be processed, and that all necessary data can be stored.

Fetuares

  • Return a list of ALL movies to the user.
  • Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user.
  • Return data about a genre (description) by name/title (e.g., “Thriller”).
  • Return data about a director (bio, birth year, death year) by name.
  • Allow new users to register.
  • Allow users to update their user info (username, password, email, date of birth).
  • Allow users to add a movie to their list of favorites.
  • Allow users to remove a movie from their list of favorites.
  • Allow existing users to deregister.
